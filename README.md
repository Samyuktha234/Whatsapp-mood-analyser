# WhatsApp Mood Analyzer

A simple WhatsApp Mood Analyzer built using C programming to analyze chat text and identify moods based on keywords and patterns.

## Features
- Analyze WhatsApp chat text
- Mood detection
- File handling
- String manipulation

## Technologies Used
- C
- Dev C++

## How to Run
1. Open in Dev C++
2. Compile the code
3. Run the program

## Learning Outcome
This project helped me improve my understanding of file handling, strings, and problem-solving in C programming.
